## `POST` /users/login <a id="login"></a>

**Request Parameters**

- Expected: Body(Object)
- Content type: application/json

**_Example Value_**

```json
{
  "email": "user@example.com",
  "password": "string"
}
```

**_Model_**

    {
    email*	    string($email)
    password*	string
                minLength: 8
    }

**Responses Parameters**

- Code: 200
- Content type: application/json

**_Example Value_**

data: {
    avatar: null
    email: email,
    firstName: string,
    id: string,
    lastName: string,
    role: 'Applicant' | 'Admin'
    token: string,
}
